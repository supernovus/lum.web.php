# lum.web.php

## Summary

Libraries for building web apps.

## Namespace

`Lum\Web`

### Classes

| Name                    | Description                                       |
| ----------------------- | ------------------------------------------------- |
| Lum\Web\Client          | Get information about the client browser.         |
| Lum\Web\Output          | Set headers for specific types of output.         |
| Lum\Web\Url             | A bunch of URL manipulation functions.            |
